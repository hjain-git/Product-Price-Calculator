# Screenshots

This directory contains screenshots of the Product Price Calculator Pro application.

To add your own screenshots:
1. Open the application in your browser
2. Take screenshots of key features
3. Save them in this directory with descriptive names like:
   - `main-interface.png` - Main calculator interface
   - `product-comparison.png` - Multiple products being compared
   - `unit-converter.png` - Unit conversion feature
   - `mobile-view.png` - Mobile responsive view
